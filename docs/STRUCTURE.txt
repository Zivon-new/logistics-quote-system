【使用流程】

1. 导入模块
   from horizontal_table_parser_v2 import HorizontalTableParserV2

2. 创建解析器
   parser = HorizontalTableParserV2(logger=logger)

3. 解析Excel
   result = parser.parse_excel('file.xlsx')

4. 查看结果
   routes = result['routes']  # 24条
   agents = result['agents']  # 35个
