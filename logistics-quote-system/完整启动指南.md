# 国际物流报价系统 - 完整启动指南

## 📦 项目概览

这是一个基于 **FastAPI + Vue 3 + MySQL** 的国际物流报价查询系统。

### 技术栈
- **后端**: FastAPI + SQLAlchemy + MySQL + JWT
- **前端**: Vue 3 + Element Plus + Vite + Axios
- **数据库**: MySQL 8.0
- **风格**: 仿照你们公司现有系统（深蓝色主题）

---

## 🚀 第一次启动（完整步骤）

### 步骤1: 准备数据库

#### 1.1 确保MySQL已安装
```bash
# 检查MySQL是否运行
mysql --version

# Windows: 在服务中启动MySQL
# 或者: net start MySQL80
```

#### 1.2 创建数据库
```bash
mysql -u root -pJHL181116
```

在MySQL中执行:
```sql
CREATE DATABASE IF NOT EXISTS price_test_v2 CHARACTER SET utf8mb4 COLLATE utf8mb4_general_ci;
SHOW DATABASES;
EXIT;
```

#### 1.3 导入表结构
```bash
# 在项目根目录执行
mysql -u root -pJHL181116 price_test_v2 < database/price_test_v2.sql
```

---

### 步骤2: 启动后端

#### 2.1 进入backend目录
```bash
cd logistics-quote-system/backend
```

#### 2.2 创建Python虚拟环境（推荐）
```bash
# Windows
python -m venv venv
venv\Scripts\activate

# 如果你看到 (venv) 前缀，说明激活成功
```

#### 2.3 安装依赖
```bash
pip install -r requirements.txt
```

**可能遇到的问题**:
- 如果报错 `pip版本过低`，先执行: `python -m pip install --upgrade pip`
- 如果某个包安装失败，尝试: `pip install <包名> -i https://pypi.tuna.tsinghua.edu.cn/simple`

#### 2.4 初始化用户表
```bash
python init_db.py
```

看到以下输出说明成功:
```
✓ 数据表创建完成
✓ 创建管理员账号: admin / admin123
✓ 创建普通用户: user / user123
✓ 数据库初始化完成！
```

#### 2.5 启动后端服务
```bash
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

**启动成功标志**:
```
INFO:     Uvicorn running on http://0.0.0.0:8000
INFO:     Application startup complete.
```

**测试后端是否正常**:
- 浏览器访问: http://localhost:8000
- 应该看到: `{"message": "欢迎使用国际物流报价系统API", ...}`
- API文档: http://localhost:8000/docs

---

### 步骤3: 启动前端

**打开新的命令行窗口**（保持后端运行）

#### 3.1 安装Node.js（如果还没安装）
- 下载: https://nodejs.org/
- 推荐版本: 18.x 或 20.x
- 安装后验证: `node --version` 和 `npm --version`

#### 3.2 进入frontend目录
```bash
cd logistics-quote-system/frontend
```

#### 3.3 安装依赖
```bash
npm install
```

**如果速度慢，使用国内镜像**:
```bash
npm config set registry https://registry.npmmirror.com
npm install
```

#### 3.4 启动开发服务器
```bash
npm run dev
```

**启动成功标志**:
```
VITE v5.0.0  ready in 500 ms

➜  Local:   http://localhost:5173/
➜  Network: http://192.168.x.x:5173/
```

#### 3.5 访问系统
浏览器打开: **http://localhost:5173**

默认登录账号:
- 管理员: `admin` / `admin123`
- 普通用户: `user` / `user123`

## 💡 快速测试

### 测试后端API

#### 1. 获取Token
```bash
curl -X POST "http://localhost:8000/api/v1/auth/login" \
  -H "Content-Type: application/json" \
  -d '{"username": "admin", "password": "admin123"}'
```

会返回:
```json
{
  "access_token": "eyJhbGc...",
  "token_type": "bearer",
  "user": {...}
}
```

#### 2. 查询报价（需要Token）
```bash
curl -X GET "http://localhost:8000/api/v1/quotes/search?起始地=深圳&目的地=新加坡" \
  -H "Authorization: Bearer <你的token>"
```

## ❓ 常见问题

### 1. 后端启动失败
**问题**: `ModuleNotFoundError: No module named 'app'`
**解决**: 确保在 `backend` 目录下运行，并已安装所有依赖

### 2. 数据库连接失败
**问题**: `Can't connect to MySQL server`
**解决**: 
- 检查MySQL是否运行
- 检查密码是否正确（JHL181116）
- 尝试: `mysql -u root -pJHL181116` 测试连接

### 3. 端口被占用
**问题**: `Port 8000 is already in use`
**解决**:
```bash
# Windows
netstat -ano | findstr :8000
taskkill /PID <进程ID> /F

# 或者更换端口
uvicorn app.main:app --reload --port 8001
```

### 4. npm install 很慢
**解决**: 使用国内镜像
```bash
npm config set registry https://registry.npmmirror.com
```
